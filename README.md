# Food-Ordering-Behavior-and-Consumer-Trends
# 🍔 Food Ordering & Customer Behavior Analysis using Power BI

## Overview

This project explores food ordering and customer behavior using Power BI. The goal is to understand customer preferences, ordering patterns, and sales performance through interactive dashboards. The insights can help businesses make informed decisions, improve customer satisfaction, and optimize operations.

## Objectives

* Analyze customer ordering trends
* Identify popular food items and categories
* Find peak ordering hours and busiest days
* Track revenue and sales performance
* Understand customer spending behavior

## Tools Used

* Microsoft Power BI
* Power Query
* DAX
* Microsoft Excel / CSV

## Dashboard Highlights

* Sales & Revenue Overview
* Top-Selling Food Items
* Customer Purchase Behavior
* Peak Ordering Time Analysis
* Order Status Distribution
* Customer Spending Trends
* Interactive Filters and Slicers

## Project Workflow

1. Collected and explored the dataset.
2. Cleaned and transformed the data using Power Query.
3. Built relationships and created DAX measures.
4. Designed interactive dashboards with meaningful KPIs.
5. Analyzed the results to identify trends and business insights.

## Key Insights

* Certain food categories generated the highest revenue.
* Most orders were placed during lunch and dinner hours.
* Returning customers contributed significantly to total sales.
* Customer spending varied across different food categories and time periods.
